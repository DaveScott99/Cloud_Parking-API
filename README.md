# Cloud_Parking-API

## Sobre o projeto

API para controle de estacionamento desenvolvida durante a realização do curso Quebec Java oferecido pela plataforma de ensino DIO (Digital Innovation One).

## Tecnologias utilizadas

### Back end

- `Java`
- `Spring Boot`
- `JPA / Hibernate`
- `Maven`
- `Banco de dados H2`
